# API Reference

Complete reference for all public APIs, interfaces, and types in the BFT consensus engine.

## Core Interfaces

### Logger

Structured logging interface with multiple severity levels.

```go
type Logger interface {
    Fatal(msg string, fields ...log.Field)
    Error(msg string, fields ...log.Field)
    Warn(msg string, fields ...log.Field)
    Info(msg string, fields ...log.Field)
    Trace(msg string, fields ...log.Field)
    Debug(msg string, fields ...log.Field)
    Verbo(msg string, fields ...log.Field)
}
```

#### Methods

##### `Fatal(msg string, fields ...log.Field)`
Log a fatal error. The program should exit after this call.
- **msg**: Error message
- **fields**: Structured logging fields

##### `Error(msg string, fields ...log.Field)`
Log a recoverable error.
- **msg**: Error message
- **fields**: Structured logging fields

##### `Warn(msg string, fields ...log.Field)`
Log a warning that may indicate future issues.
- **msg**: Warning message
- **fields**: Structured logging fields

##### `Info(msg string, fields ...log.Field)`
Log informational messages for progress tracking.
- **msg**: Info message
- **fields**: Structured logging fields

##### `Trace(msg string, fields ...log.Field)`
Log execution order for debugging.
- **msg**: Trace message
- **fields**: Structured logging fields

##### `Debug(msg string, fields ...log.Field)`
Log detailed debugging information.
- **msg**: Debug message
- **fields**: Structured logging fields

##### `Verbo(msg string, fields ...log.Field)`
Log extremely detailed information.
- **msg**: Verbose message
- **fields**: Structured logging fields

### BlockBuilder

Interface for application-specific block construction.

```go
type BlockBuilder interface {
    BuildBlock(ctx context.Context, metadata ProtocolMetadata) (VerifiedBlock, bool)
    IncomingBlock(ctx context.Context)
}
```

#### Methods

##### `BuildBlock(ctx context.Context, metadata ProtocolMetadata) (VerifiedBlock, bool)`
Build a new block with transactions.
- **ctx**: Context for cancellation
- **metadata**: Protocol metadata for block construction
- **Returns**: Built block and success flag

##### `IncomingBlock(ctx context.Context)`
Signal that a block should be built.
- **ctx**: Context for cancellation

### Storage

Persistent storage for blocks and finalization proofs.

```go
type Storage interface {
    Height() uint64
    Retrieve(seq uint64) (VerifiedBlock, Finalization, bool)
    Index(ctx context.Context, block VerifiedBlock, certificate Finalization) error
}
```

#### Methods

##### `Height() uint64`
Get the current blockchain height.
- **Returns**: Current block height

##### `Retrieve(seq uint64) (VerifiedBlock, Finalization, bool)`
Retrieve block and finalization at sequence number.
- **seq**: Sequence number to retrieve
- **Returns**: Block, finalization certificate, and success flag

##### `Index(ctx context.Context, block VerifiedBlock, certificate Finalization) error`
Store a finalized block.
- **ctx**: Context for cancellation
- **block**: Verified block to store
- **certificate**: Finalization certificate
- **Returns**: Error if storage fails

### Communication

P2P communication layer for consensus messages.

```go
type Communication interface {
    Nodes() []NodeID
    Send(msg *Message, destination NodeID)
    Broadcast(msg *Message)
}
```

#### Methods

##### `Nodes() []NodeID`
Get all participating nodes in the epoch.
- **Returns**: List of node IDs

##### `Send(msg *Message, destination NodeID)`
Send a message to a specific node.
- **msg**: Message to send
- **destination**: Target node ID

##### `Broadcast(msg *Message)`
Broadcast message to all nodes except self.
- **msg**: Message to broadcast

Aggregate BLS signatures into quorum certificates.


#### Methods

##### `Aggregate([]Signature) (QuorumCertificate, error)`

### WriteAheadLog

Optional WAL for crash recovery.

```go
type WriteAheadLog interface {
    Append([]byte) error
    ReadAll() ([][]byte, error)
}
```

#### Methods

##### `Append([]byte) error`
Append a record to the WAL.
- **record**: Record bytes to append
- **Returns**: Error if write fails

##### `ReadAll() ([][]byte, error)`
Read all records from the WAL.
- **Returns**: Array of record bytes and error

## Core Types

Unique identifier for a consensus node.

```go
type NodeID []byte
```

#### Methods

String representation of node ID.

##### `Equals(other NodeID) bool`

### BlockHeader

Succinct block representation.

```go
type BlockHeader struct {
    Round     uint64
    Author    NodeID
    Parent    BlockHeader

}
```

#### Fields

- **Round**: Consensus round number
- **Author**: Block proposer node ID
- **Parent**: Parent block header
- **Timestamp**: Block creation timestamp

### Signature

Individual signature with signer identification.

```go
type Signature struct {
    Signer NodeID
    Value  []byte
}
```

#### Fields

- **Signer**: Node ID of signature creator
- **Value**: Signature bytes

### Message

Consensus protocol message.

```go
type Message struct {
    BlockMessage      *BlockMessage
    VoteMessage       *VoteMessage
    EmptyVoteMessage  *EmptyVoteMessage
    FinalizeMessage   *FinalizeMessage
    ReplicationRequest *ReplicationRequest
    ReplicationResponse *ReplicationResponse
}
```

#### Message Types

##### BlockMessage
Contains a proposed block.

##### VoteMessage
Contains a notarization vote.

##### EmptyVoteMessage
Timeout vote for liveness.

##### FinalizeMessage
Contains a finalization vote.

##### ReplicationRequest
Request for missing blocks.

##### ReplicationResponse
Response with requested blocks.

## Usage Examples

### Creating an Epoch

```go
config := EpochConfig{
    ID:                 NodeID("node-1"),
    Epoch:              1,
    StartTime:          time.Now(),
    MaxProposalWait:    10 * time.Second,
    MaxRebroadcastWait: 30 * time.Second,
    Logger:             logger,
    Signer:             signer,
    Verifier:           verifier,
    // ... other components
}

epoch, err := NewEpoch(config)
if err != nil {
    log.Fatal("Failed to create epoch", err)
}

ctx := context.Background()
if err := epoch.Start(ctx); err != nil {
    log.Fatal("Failed to start epoch", err)
}
```

### Implementing BlockBuilder

```go
type MyBlockBuilder struct {
    txPool *TransactionPool
}

func (b *MyBlockBuilder) BuildBlock(
    ctx context.Context,
    metadata ProtocolMetadata,
) (VerifiedBlock, bool) {
    select {
    case <-ctx.Done():
        return nil, false
    case txs := <-b.txPool.Ready():
        block := createBlock(txs, metadata)
        return block, true
    }
}

func (b *MyBlockBuilder) IncomingBlock(ctx context.Context) {
    select {
    case <-ctx.Done():
        return
    case <-b.txPool.Signal():
        return
    }
}
```

## Thread Safety

All public methods on the following types are thread-safe:
- `Epoch`
- `TimeoutHandler`
- `ReplicationState`

The following interfaces must be implemented thread-safely:
- `Storage`
- `Communication`
- `WriteAheadLog`

## Performance Considerations

### Signature Verification
- Batch verify signatures when possible
- Cache verification results
- Use parallel verification for large sets

### Storage Operations
- Batch writes for efficiency
- Use async indexing where appropriate
- Implement proper caching layers

### Network Communication
- Batch messages to reduce syscalls
- Implement proper flow control
- Use compression for large messages

## Migration and Compatibility

### Version Compatibility
- Protocol messages are versioned
- Backward compatibility for one major version
- Forward compatibility via unknown field handling

### Storage Migration
- Blocks are immutable once finalized
- Migration tools provided for storage backend changes
- WAL format is stable across versions
